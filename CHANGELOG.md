# Changelog

All notable changes to the Beyond MVG research outputs are documented here.

## [1.1.0] — 2025-12-02 (Revised Edition)

### Changed
- **Governance Measurement Framework**: Removed references to Blockfrost API for verification; cardano-db-sync is now the sole primary data source
- **Governance Metrics Manual**: Updated data source references to align with Measurement Framework
- **State of Governance Report Outline**: Minor consistency updates across sections

## [1.0.0] — 2025-12-02 (Initial Release)

### Added
- Governance Measurement Framework (M1-1D)
- Governance Metrics Manual (M1-1D)
- State of Governance Report Outline (M1-1D)
- Defined 12 core governance metrics across 4 categories:
  - Ada Holder Metrics (Legitimacy & Participation)
  - DRep Metrics (Decentralization & Accountability)
  - SPO Metrics (Security & Oversight)
  - Constitutional Committee Metrics (Constitutional Adherence & Veto Oversight)
