# Contributing to Beyond MVG

Thank you for your interest in contributing to the State of Cardano Governance project. This document outlines how you can help improve the governance measurement framework, metric definitions, tooling, and data outputs.

---

## Ways to Contribute

### Reporting Issues
If you find errors, inconsistencies, or outdated information in any of the documents:
1. Open an [Issue](../../issues) using the **Bug Report** template
2. Describe the problem clearly and reference the specific document and section
3. If possible, suggest a correction

### Suggesting Improvements
To propose new metrics, updates to calculation logic, or additions to the report outline:
1. Open an [Issue](../../issues) using the **Feature Request** template
2. Explain the rationale and how the change aligns with the governance measurement goals

### Submitting Code or Data
For contributions to the `code/` or `data/` folders (scripts, SQL queries, datasets):
1. Fork the repository
2. Create a new branch: `git checkout -b your-feature-name`
3. Make your changes
4. Submit a Pull Request with a clear description of what was added or changed

---

## Document Versioning

When updating PDFs in `docs/`, please:
- Keep the same filename to maintain stable links
- Update [CHANGELOG.md](CHANGELOG.md) with a summary of what changed and the date

---

## Code of Conduct

This project follows the Cardano community's standards of respectful, constructive collaboration. Contributions that are disrespectful, off-topic, or in bad faith will not be accepted.

---

## Questions?

Open an issue or reach out through the Cardano governance forums.
