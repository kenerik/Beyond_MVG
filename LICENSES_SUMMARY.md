**Status:** ✅ **ALLOWED**

- You must include the Apache 2.0 license and attribution

**Status:** ❌ **NOT ALLOWED** (without permission)

- Must contact IOG for commercial licensing

### Scenario 3: Using Data for Non-Profit Research
**Status:** ✅ **ALLOWED**

- Can use for non-profit analysis
- Must provide proper attribution to IOG

**Status:** ✅ Code can be commercial; data usage depends on purpose
